# Dotfiles managed with chezmoi

This repository contains my dotfiles, managed with [chezmoi](https://www.chezmoi.io/).

## Installation

1. Install chezmoi:

```sh
sh -c "$(curl -fsLS get.chezmoi.io)" 
```

1. Setup dotfiles:

```sh
chezmoi init --apply git@github.com:StevenSermeus/dotfiles.git
```
